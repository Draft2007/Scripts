---ElJefe XML Server---

1- Server's certificate and key is automatically created the first time you run the server. 
Also is created CA's certificate and key. 

2- The certificate is created with the XML server's IP (you must set the IP in settings.py), 
if you change the IP you must create a new Server's certificate.  

NOTES:
(1) If you want reset server's certificate and key you must delete the files in "/certs" 
folder.
